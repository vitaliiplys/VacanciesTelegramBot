# 🤖 Job Finder Telegram Bot

![1](./Images/1.png)

## 📌 Description

Job Finder Bot — це Telegram — бот для пошуку IT-вакансій за рівнем досвіду (Junior, Middle, Senior).
Бот агрегує вакансії з dou.ua та djinni.co та дозволяє швидко отримати актуальні пропозиції прямо в Telegram.

## 🚀 Features

- 🔎 Пошук вакансій за рівнем:
    - Junior (0–1 рік)
    - Middle (1–3 роки)
    - Senior (3+ роки)
- 📊 Фільтрація за досвідом
- 🛠 REST-архітектура
- ⚙️ Spring Boot application

## 🏗 Tech Stack

- Spring Boot
- Telegram Bot API
- Maven
- Jsoup (для парсингу)
